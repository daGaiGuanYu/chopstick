# 筷子
你值得拥有快乐，请不要放弃。

## 定位
前后端分离后的“后端”的工具集（基于 node），用来写接口

## 优点
+ 简单：源码简单、使用简单、文档简单
+ 单纯
+ AOP
+ 中文本位，系统运行日志、报错、文档均为中文

## 原则
+ 简单
+ 进步大于稳定
+ 适当拓展性
+ [锋利刀子](https://rubyonrails.org/doctrine/#provide-sharp-knives)

## 适用人群
+ 对 node 原生 api 有一定了解
+ 个人开发者
+ 小型团队
